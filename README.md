## Всем привет! Меня зовут Константин
Здесь я размещаю свои проекты в области дата инжиниринга, аналитики данных. Также интересуюсь web-технологиями в ключе закрытия цепочки от сбора данных до их визуализации.

Развиваюсь в сфере **Data Engineering** и **Data Analytics**.

### 🛠️ Навыки и инструменты
* **Данные и аналитика:** Python (Pandas, NumPy), SQL (ClickHouse / PostgreSQL), расчет продуктовых метрик (DAU/MAU, Retention, воронки конверсии, ARPPU), продуктовая аналитика, основы A/B-тестирования.
* **Другие навыки в IT-сфере:** 
  - Системное администрирование и ОС: Linux, Windows Server, Active Directory (работа в домене), Групповые политики (GPO), управление правами доступа.
  - Деплой и инфраструктура: Развертывание ПО на серверах, автоматизация задач, основы Docker.
  - Сети и безопасность: Базовая настройка сети (IP, DNS, DHCP), диагностика сетевых подключений, работа с ЭЦП (КриптоПро, Рутокен).
  - Поддержка и процессы: Написание технической документации (инструкции, базы знаний), регламентация ИТ-процессов.

### 📂 Мои проекты
* **[Дашборд анализа вовлеченности и удержания пользователей](https://github.com/constantinepyro666/analytics-project)** — Аналитический дашборд поведения пользователей мобильного приложения: DAU, retention и воронка конверсии пользователя на разных платформах (iOS, Android, Web).
* **[ETL-пайплайн мониторинга цен товаров](https://github.com/constantinepyro666/price-tracker)** — Проект реализует автоматический сбор, хранение и визуализацию цен товаров с использованием PostgreSQL, Docker и Streamlit.
